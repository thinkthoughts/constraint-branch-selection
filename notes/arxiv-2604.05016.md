# arXiv:2604.05016 — Constraint Branch Selection Note

## Reference

- Title: (Nomura, Totani, 2026)
- Topic: Galactic halo γ-ray excess and minimal dark matter model
- Link: https://arxiv.org/abs/2604.05016

---

## Core structure

This paper is a clear example of:

signal → degeneracy → constraints → narrowed parameter space (≠ unique solution)

---

## Observational input

- Galactic halo γ-ray excess
- Can be explained by multiple astrophysical and particle models

→ **signal alone is not identifying**

---

## Model proposal

- Electroweak doublet dark matter  
- Higgs portal interaction  
- Mass range: ~400–800 GeV  
- Dominant annihilation: W / Z bosons  

---

## Constraint stack

The model is tested against:

- relic abundance (thermal freeze-out)
- indirect detection (γ-rays)
- direct detection (inelastic scattering, Δm ~ 100 keV)
- astrophysical bounds (e.g. dwarf galaxies)

---

## Result

- parameter space is significantly **reduced**
- viable region remains within a constrained range
- **degeneracy is not fully removed**

→ multiple parameter combinations survive

---

## Key distinction

The paper demonstrates:

- **fit**: model reproduces γ-ray excess  
- **constraint consistency**: model survives multiple tests  
- **identifiability**: NOT achieved  

---

## Interpretation

This is a case of:

> constraint-based branch selection without full identifiability

The constraints act as a **selection filter**, but not a **unique selector**.

---

## Branch structure

- initial signal → large continuous solution space  
- added constraints → reduced to a bounded region  
- remaining space → still contains multiple viable branches  

---

## Identifiability gap

Even after constraints:

- parameter degeneracy persists  
- multiple branches remain observationally indistinguishable  

→ solution is **stable but not unique**

---

## What would resolve it?

Identifiability likely requires:

- additional independent observables  
- orthogonal constraints  
- higher-resolution measurements  

Examples:

- combined indirect + direct detection signals  
- time-dependent or spatial structure in γ-ray data  
- improved sensitivity to inelastic transitions  

---

## General lesson

> constraint stacking ≠ unique identification

A system can be:

- well-fit  
- stable  
- physically consistent  

and still:

- non-identifiable  

---

## Mapping to this repo

This paper corresponds to:

- multi-constraint filtering  
- surviving parameter islands  
- unresolved branch degeneracy  

It is a real-world instance of the toy structure:

observable → constraints → surviving branches → (missing step: unique selection)
